# ScholarScout – Multi-Agent Scholarship Discovery System

ScholarScout is a lightweight multi-agent AI system that helps students:

✓ Find relevant scholarships  
✓ Check eligibility  
✓ Understand reasons for acceptance/rejection  
✓ Get a personal step-by-step application plan  

This project is simplified to run without external libraries while demonstrating:

- Multi-Agent Architecture  
- Sequential Agent Pipeline  
- Context Passing  
- Modular Components  
- Realistic Scholarship Evaluation  

## How to Run

```bash
python main.py
